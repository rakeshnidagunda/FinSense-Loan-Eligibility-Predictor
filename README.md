# FinSense--Loan-Eligibility-Predictor
End-to-end AI-powered personal loan eligibility predictor built with Python, scikit-learn, and Flask. Achieves 94.8% CV accuracy using Gradient Boosting, with full EDA, model comparison, explainable AI outputs (SHAP-style feature importance), and deployment-ready REST API. Designed for real-time credit risk scoring in fintech environments.
